For Problem 1:

To compile all executables, run "make all"

To run an executable, run "make target=prog args="num_threads problem_size"

where "prog" is one of the executables:

"am" - part a, monte carlo
"al" - part a, liebniz
"bm" - part b, monte carlo
"bl" - part b, liebniz

example run command: make run target=am args="1 10000"
